# Enterprise WorkHub Project

This project is a Spring Boot backend for WorkHub, a multi-tenant SaaS application for managing organizations and their users. It is being developed as an enterprise application project with a focus on authentication, tenant-aware design, and backend architecture.

## Tech Stack

- Java 25
- Maven
- Spring Boot 4
- OpenRewrite
- PostgreSQL

## Project Idea

WorkHub is a backend system where each tenant represents an organization. Each organization has its own users, and the platform is being built to support secure authentication, tenant isolation, and enterprise-style backend features over multiple project phases.
